# Web Effect Studio

前端炫酷效果实验室 — 在线编辑、预览、分享 Web 动画效果。

## 功能

- 🎨 **效果画廊** — 29+ 内置效果，悬停实时预览
- ⌨️ **Monaco 编辑器** — 专业代码编辑 + 代码片段快捷插入
- 🖥️ **实时预览** — 所有更改即时呈现
- 🎮 **小游戏 & UI 组件** — 贪吃蛇、乒乓球、打砖块、数独等
- 💾 **本地保存** — 自动保存 + 手动保存，localStorage 持久化
- 🔗 **分享链接** — 一键生成可分享链接，无需注册
- ⌨️ **快捷键** — Ctrl+S 保存 / Ctrl+Enter 刷新 / Ctrl+1/2/3 切换标签
- 📺 **全屏预览** — 专注效果，隐藏编辑器
- 🔀 **Fork** — 一键复制内置效果到个人项目
- 🎲 **随机效果** — 随机跳转到任意效果

## 快速开始

```bash
npm install
npm run dev
```

打开 http://localhost:5173

## 技术栈

React 18 + Vite + TypeScript + Monaco Editor + TailwindCSS + Zustand

## 构建

```bash
npm run build
```

输出到 `dist/`
