# Web Effect Studio 设计系统

> 融合阿里 Ant Design、腾讯 TDesign、字节 Arco Design、华为 DevUI 等大厂设计规范

## 📁 目录结构

```
design-system/
├── tokens/          # 设计令牌（颜色、字体、间距、阴影等）
├── components/      # 组件规范（按钮、输入框、卡片等）
├── patterns/        # 设计模式（表单、列表、导航等）
├── resources/       # 资源文件（图标、插图、字体）
└── references/      # 参考文档（大厂规范链接）
```

## 🎨 设计原则

1. **一致性** - 保持视觉和交互的统一
2. **可用性** - 以用户为中心的设计
3. **美观性** - 现代、简洁、专业
4. **可扩展性** - 易于维护和迭代
