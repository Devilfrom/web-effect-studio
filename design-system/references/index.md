## 🇨🇳 国内大厂

### 阿里巴巴

  - 企业级 UI 设计语言和 React 组件库
  - 涵盖视觉、交互、前端实现规范
  - Figma 设计资源齐全

  - 数据可视化解决方案

- **WeUI** - https://weui.io/
  - 微信原生视觉体验的基础样式库

### 字节跳动

## 🎯 设计原则总结

### 一致性
- 统一的色彩、字体、间距系统
- 组件样式和交互保持一致
- 跨平台体验统一

### 可用性
- 清晰的视觉层级
- 直观的操作流程
- 及时的反馈机制

### 可访问性
- 足够的色彩对比度
- 支持键盘导航
- 适配屏幕阅读器

### 可扩展性
- 设计令牌化，易于主题定制
- 组件模块化，支持组合
- 文档完善，易于上手
